title: "How to View Campaign Analytics in bitCRM"
---

This guide explains how to access and interpret the performance data of your WhatsApp campaigns in bitCRM. Monitoring these metrics is essential to understanding.

## Step-by-Step Guide

1. **Navigate to Campaigns:** On the left-hand sidebar Navigate to the bitCRM, click on **Campaign.**

   ![Screenshot2025 11 27100421 Pn](/images/Screenshot2025-11-27100421.png)
1. On the Campagin Dashboard. **Select Your Campaign:**
   - Find the specific campaign you want to analyze in the list.
   - Click on the **the box on the left of the Campaign Name**

![Screenshot2025 11 27100548 Pn](/images/Screenshot2025-11-27100548.png)

3. Click Export Data

   ![Screenshot2025 11 27100841 Pn](/images/Screenshot2025-11-27100841.png)
4. The result will be in the format of CSV.

   | No.             | Description                                                                                        |
   | --------------- | -------------------------------------------------------------------------------------------------- |
   | No              | The row number for reference.                                                                      |
   | Campaign Name   | The internal name you gave this specific broadcast or automation.                                  |
   | Channel         | The platform used to send the message                                                              |
   | Receiver Number | The phone number the message was sent to.                                                          |
   | Receiver Name   | The customer name associated with that number in your database.                                    |
   | Status          | The current state of the message (e.g., _Sent_, _Delivered_, _Read_, or _Failed_).                 |
   | Send Date       | The exact date and timestamp when the system attempted to send the message.                        |
   | Message Body    | The actual content/text that was sent to the customer.                                             |
   | Replied Message | If the customer responded, their text will appear here. **Check this regularly for leads.**        |
   | Error Message   | If the Status is "Failed," this column explains why (e.g., _Invalid Number_, _Template Rejected_). |
