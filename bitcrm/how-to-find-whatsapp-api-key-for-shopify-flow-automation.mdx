---
title: "How To Find WhatsApp API Key For Shopify Flow Automation"
---

Easily locate and use your WhatsApp API key to automate workflows within Shopify, making customer communication faster and more streamlined. This guide walks you through every step from copying your API key to integrating it with Shopify flow automation.

Step 1 : Go to your profile and click the triple dots. After clicking it there will be a pop out of "Settings and Logout" Choose settings and go to advance settings.

And After copying the API Key please go to Shopify. After going to [Shopify](https://www.shopify.com/) please go the the App section and search Flow
