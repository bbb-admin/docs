---
title: "How to Schedule Steps in an Automation"
---

Creating automations in bitCRM allows you to schedule and manage tasks more effectively, ensuring that your customer communication is timely and relevant. This guide will walk you through the steps to schedule actions in your automation scenarios.

## Step 1: Access the bitCRM Automation Dashboard

Go to the [bitCRM Automation dashboard](https://app.bitbybit.studio).

## Step 2: Create a New Scenario

Click **"Create new scenario"**.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Automation-Template-Dashboard.png"
  alt="WhatsApp API Automation Template Dashboard"
/>

## Step 3: Define the Trigger

Choose the trigger based on your needs. For example:

- **Shopify Trigger**:

  - `Order Fulfilled`: Trigger a message to confirm the order's delivery.
  - `Order Created`: Send a confirmation message for newly created orders.
  - `Order Paid`: Send a payment confirmation message to the buyer.
  - `Order Created (unpaid)`: Notify the buyer that the order was created but not yet paid.
  - `Abandoned Cart`: Send a reminder to customers to complete their purchase.

- **bitLogin Trigger**:
  - `Login`: Trigger a welcome message when the user logs in.
  - `Sign Up`: Send a welcome message for new users.

Click **"+"** to add steps.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Automation-Template-Dashboard.png"
  alt="Add steps"
/>

## Step 4: Add Actions

Select and define actions such as:

- **Communication Channel**: Choose where to send the message.
- **Product Review**: Ask customers for product reviews.
- **Time Delay**: Set a delay before the next action.
- **Conditions**: Specify conditions to control how the automation flows.

## Step 5: Save the Scenario

Click **"Save"** and then **"Save automation"**.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Automation-Template-Dashboard.png"
  alt="Save automation"
/>
