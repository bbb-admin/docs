---
title: "How to Create a Campaign with Button Options"
---

Creating a campaign with button options in bitCRM helps you engage customers directly through WhatsApp and makes interactions seamless. This feature allows you to provide quick replies, collect data through forms, or direct users to specific websites. Follow these steps to create a campaign with various button types.

## **Step 1:** Go to the [WhatsApp API settings](https://app.bitbybit.studio/misc/whatsapp-api-settings) dashboard.

## **Step 2:** Before using Button Options, ensure you have created a campaign template first, see template below for guidance.

**1.** How to Create a Message Template Campaign for WhatsApp Cloud API.

**2.** How to Create a Message Template Automation for WhatsApp Cloud API.

**3.** How to Create a Chat Initiation Template.

## **Step 3:** Click the toggle to green then select the preferred dropdown button type.
