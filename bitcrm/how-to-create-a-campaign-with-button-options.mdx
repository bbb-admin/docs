## Step 1: Go to the WhatsApp API Settings Dashboard

Navigate to the [WhatsApp API settings dashboard](https://app.bitbybit.studio) in bitCRM.

<img
  src="/images/whatsapp-api-settings-dashboard.png"
  alt="WhatsApp API Settings Dashboard"
  className="rounded-lg"
/>

## Step 2: Create a Campaign Template

Before adding button options, ensure you have created a campaign template. For guidance, refer to the following articles:

- [How to Create a Message Template Campaign for WhatsApp Cloud API](https://www.bitbybit.studio/knowledge-base/how-to-create-a-message-template-campaign-for-whatsapp-cloud-api)
- [How to Create a Message Template Automation for WhatsApp Cloud API](https://www.bitbybit.studio/knowledge-base/how-to-create-a-message-template-automation-for-whatsapp-cloud-api)
- [How to Create a Chat Initiation Template](https://www.bitbybit.studio/knowledge-base/how-to-create-a-chat-initiation-template)

## Step 3: Enable Button Options

Toggle the button to green to enable button options, then select your preferred button type from the dropdown menu.

### Button Types:

1. **Reply Button**: Allows instant replies. Example triggers: "Main Menu," "Promo."

   - Fill in the "Reply button text."

   <img
     src="/images/reply-button.png"
     alt="Reply Button"
     className="rounded-lg"
   />

2. **WhatsApp Form**: Collect data like surveys or reservations within WhatsApp.

   - Select "WhatsApp form" from the dropdown or create a new WhatsApp form. For more information, refer to [How to Create a WhatsApp Form](https://www.bitbybit.studio/knowledge-base/how-to-create-a-whatsapp-form).
   - Fill in the "Button text."

   <img
     src="/images/whatsapp-form.png"
     alt="WhatsApp Form"
     className="rounded-lg"
   />

3. **Call to Action**: Direct users to a website or landing page.

   - Select from "Web URL," "Specific product," or "Specific collections."
   - Fill in the "URL" and "Button text."

   <img
     src="/images/call-to-action.png"
     alt="Call to Action"
     className="rounded-lg"
   />

4. **WhatsApp Catalog**: Present your product catalog to customers within WhatsApp.

   - Add "Button Text" and ensure you have a catalog set up. For guidance, refer to [How to Create and Manage a Collection in Your Catalog](https://faq.whatsapp.com/).

   <img
     src="/images/whatsapp-catalog.png"
     alt="WhatsApp Catalog"
     className="rounded-lg"
   />

## Step 4: Save the Template

After configuring your button options, click "Save template" to apply the changes.
