---
title: "How to Set Up a Trigger"
---

Setting up a trigger in bitCRM allows you to automate communication and manage routine tasks, like order confirmations and reminders. By following these steps, you can create a customized automation that fits your business needs.

## Step 1: Access the bitCRM Automation Dashboard

Go to the [bitCRM Automation dashboard](https://app.bitbybit.studio).

## Step 2: Create a New Trigger Scenario

Click **"Create new scenario"**.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Automation-Template-Dashboard.png"
  alt="WhatsApp API Automation Template Dashboard"
/>

## Step 3: Define the Trigger

Select a trigger that best fits your needs. Some examples include:

- **Order Fulfilled**: Trigger a confirmation message when the order is delivered.
- **Order Created**: Trigger a message when an order is created.
- **Order Paid**: Notify the customer when payment is processed.
- **Abandoned Cart**: Send a reminder if a customer has abandoned their cart.

## Step 4: Configure Automation Steps

Once the trigger is defined, add the necessary actions and steps to automate the flow of communication.
