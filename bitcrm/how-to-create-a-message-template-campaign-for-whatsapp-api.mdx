## How to Create a Message Template Campaign for WhatsApp API

Craft reusable message templates with dynamic variables to simplify communication and personalize your WhatsApp API campaigns. Perfect for reminders, promotions, or updates, templates ensure consistency and save time.

### Step 1: Go to the WhatsApp API Campaign Template Dashboard

Navigate to the [WhatsApp API Campaign Template dashboard](https://app.bitbybit.studio) and click **"Create new template"**.

<img
  src="/images/create-new-template.png"
  alt="Create New Template"
  className="rounded-lg"
/>

### Step 2: Fill in the Required Template Fields

Enter the necessary information for your template, including the template name and language.

<img
  src="/images/fill-template-fields.png"
  alt="Fill Template Fields"
  className="rounded-lg"
/>

### Step 3: Compose Your Message

In the "Message" field, input your message content. To add dynamic variables, click **"Add variable +"**, then select **"Variable"** from the dropdown.

For example:

`Hello {{1}}, your order {{2}} is on its way and will arrive on {{3}}.`

This allows you to personalize each message with customer-specific information.

<img
  src="/images/compose-message.png"
  alt="Compose Message"
  className="rounded-lg"
/>

### Step 4: Upload an Image (Optional)

To enhance your message, you can upload an image. Click **"Upload image"**, select your image file, and click **"Save template"**.

<img src="/images/upload-image.png" alt="Upload Image" className="rounded-lg" />

### Step 5: Preview Your Message

On the right sidebar, preview your message to ensure it appears as expected.

<img
  src="/images/preview-message.png"
  alt="Preview Message"
  className="rounded-lg"
/>

Once your message template is created, you can use it in campaigns, automations, or chat initiations. For guidance on creating a campaign, refer to [How to Create a New WhatsApp API Campaign](https://www.bitbybit.studio/knowledge-base/how-to-create-a-new-whatsapp-api-campaign).

## How to Add an Image to Your Campaign

Adding images to your campaigns can significantly increase engagement. Follow these steps to add an image to your WhatsApp campaign:

1. Navigate to your campaign settings in bitCRM.
2. In the message field, click **"Upload Image"** to select an image from your device.
3. After selecting the image, click **"Save"**.

## How to Create a Campaign with Button Options

Adding button options to your campaign allows you to create interactive messages. You can include buttons like "Reply," "Call," or "Visit Website" to engage customers more effectively.

1. While creating your message, click **"Add Button"**.
2. Choose a button type (e.g., Reply, Call to Action) and configure the button action.

## How to Edit or Cancel an Already Scheduled Campaign

If you need to make changes or cancel a scheduled campaign, follow these steps:

1. Go to the **Campaign Dashboard**.
2. Find the scheduled campaign and click **Edit** or **Cancel**.
3. For editing, make the necessary changes and click **Save**.
