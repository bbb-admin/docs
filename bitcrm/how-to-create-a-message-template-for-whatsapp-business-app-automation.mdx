---
title: "How to Create a Message Template for WhatsApp Business App Automation"
---

Automating WhatsApp messages in bitCRM helps streamline customer communication by sending real-time updates for orders, payments, and abandoned carts. This guide walks you through creating a message template and setting up automation to improve engagement and efficiency.

## **Step 1: Access the bitCRM Automation Dashboard**

- Go to the [bitCRM Automation](https://app.bitbybit.studio/bitcrm/automation-new) dashboard.

## **Step 2: Create a New Scenario**

- Click "Create new scenario"

![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/66f5090a8f80679c4af29197_AD_4nXcOXzD_NSvrJFws7xX3NQQCpMRHD1KZMcCbEX1zfKpk231kvVD1vILvnpijNbWqdGsGRKYnQPpibBzgSfjo39CDgeM0QB70rTXlRpxaxUvNFLi3xxr9v-OZwaKQ-Cl97Y7FfoTr_IxN9JfWiBJIXaOfNTY.png)

Click "Create new scenario"## **Step 3: Create a Scenario**

- Create an automation scenario. Read [How to Create an Automation](https://knowledge.bitbybit.studio/bitcrm/how-to-create-an-automation) for more details.
- Select and choose "Shopify Trigger"

![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/6706eaba8cac047bba2e1eae_AD_4nXcU5zetM7rnki6isxf9lWYGH-ob61aCe-gPllLE2psWLWuO0kmq5f_UswKCRKWfebmGLWyf1u4e_1wqx8GjXeqKT20cw96CeJsmnN_DELr2eRbEMikX22EUKqh3yRlNWKguWMG5Ucnnbvavt31dxXB0DNQe.png)

Select and choose "Shopify Trigger"1. **Order Fulfilled**: Send a message to confirm that the order has arrived at its destination.

2. **Order Created**: Send an order-created confirmation message to a buyer.
3. **Order Paid**: Send a message to confirm that the buyer's payment has been successfully processed.
4. **Order Created (unpaid)**: Send a message notifying the buyer that the order has been created but is awaiting payment.
5. **Abandoned Checkout**: Send a reminder so customers will buy the items in the cart.

- Click "\+" to add steps.

![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/66f5149b8b53a8e2864a5f52_AD_4nXfAYbhkMrJ8uWpICPRUjmj24QuR_LMVY9X_yjXEynvxXYX4YjNqn1CHAqXNwmTI9Plj_Ih_BieeibDih_XHQ3JBe9l4b9VHqkWtYmiveSdGHlYXA5Of6J7fUJPFO5_pwkf72xRTh2Kr2kLex2gaqRFiQkY.png)

Click "\+" to add steps\* Select “WhatsApp Business App” and click “Send a message”.

![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/67caa8a36f5ea848efa21be7_AD_4nXc3N8OKg02MTU2mStDWsJNanC6Gb9t9k1mMFRvgd62D6g2tQCT2ElMkml9jOUjM5fNQtkBghwM53YzSmhLt9XSnAB1lPA3t00_DOmZ98_2WyUZbtikNFoWDu5EGYsy9onal6e7zDQ.png)

Select “WhatsApp Business App” and click “Send a message”\* Fill in the required fields and “Save”.

![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/67caa8a32ca00d78c4a744f6_AD_4nXcNotiE7ornXBY-HSn1EViQSqdu-nk3Qi_r-adHxrue6GtVZ_1t1SNGqfulyfeX4S3yNyauNCw3s5s_DBlS2A6iRxyiWcMKLUmpzxWBRbZO6E0aXaZ8Ir-K30P_wOJ2sStYPnNjqA.png)
