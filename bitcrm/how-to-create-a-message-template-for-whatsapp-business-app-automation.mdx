---
title: "How to Create a Message Template for WhatsApp Business App Automation"
---

Creating an automation message template for WhatsApp Business App helps businesses streamline communication with customers using automated messages. This guide will walk you through the process of creating a message template and setting up triggers and actions to automate communication with customers.

## Step 1: Access the WhatsApp Business API Dashboard

Log in to your [WhatsApp Business API Dashboard](https://www.bitbybit.studio).

## Step 2: Create a New Message Template

Click **"Create Template"** to start a new message template.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Automation-Template-Dashboard.png"
  alt="WhatsApp API Automation Template Dashboard"
/>

## Step 3: Define Your Message Template

Fill out the template with the content you want to send. Include text, images, or interactive elements.

- **Text**: Write your message clearly and concisely.
- **Images**: Add relevant images to enhance the message.
- **Buttons**: Use buttons for quick replies or action triggers.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Message-Template.png"
  alt="WhatsApp API Message Template"
/>

## Step 4: Set the Message Template Category

Choose the appropriate category for your template:

- **Account Update**: Send notifications about account changes or updates.
- **Shipping Update**: Notify customers about their shipping status.
- **Order Update**: Inform customers about their order status.
- **Payment Update**: Notify customers about payment confirmations.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Message-Template-Category.png"
  alt="WhatsApp API Message Template Category"
/>

## Step 5: Test Your Template

Before using the template, test it to ensure it works correctly with the WhatsApp API.

Click **"Send Test"** to verify the message format and delivery.

## Step 6: Save and Use the Template

Once the template is verified, click **"Save"** and use it for your automation workflows.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Save-Template.png"
  alt="Save WhatsApp API Template"
/>
