---
title: "How to Create a Message Template for WhatsApp API Automation"
---

Creating a message template for WhatsApp API automation allows businesses to streamline communication with customers. Follow this guide to set up and customize your template.

## Step 1: Access the WhatsApp API Automation Dashboard

Go to the [WhatsApp API Automation dashboard](https://www.bitbybit.studio).

## Step 2: Create a New Template

Click **"Create new template"**.

<img
  src="https://www.bitbybit.studio/wp-content/uploads/2023/07/WhatsApp-API-Automation-Template-Dashboard.png"
  alt="WhatsApp API Automation Template Dashboard"
/>
## Step 3: Define the Template Structure

Choose a template structure based on your needs:

- **Text Message**: Simple text-based template for quick communication.
- **Media Message**: Include images, videos, or files in your template.
- **Button Template**: Add interactive buttons like "Call to Action" or "Quick Reply".

Click **"+"** to add elements.

<img
  src="https://www.bitbybit.studio/images/add-template-elements.png"
  alt="Add template elements"
/>

## Step 4: Customize the Message

Edit the message content to include dynamic variables like customer name, order details, or product information:

- **Personalization**: Use variables like `{{customer_name}}` to personalize the message.
- **Language**: Choose the language for the message content.
- **Template Preview**: Check how your template will look before sending.

## Step 5: Save and Test the Template

Click **"Save"** to store your template, then **"Test Template"** to ensure it works properly.

<img
  src="https://www.bitbybit.studio/images/save-template.png"
  alt="Save template"
/>
