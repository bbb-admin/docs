---
title: "How to Create a Message Template for WhatsApp API Automation"
---

Message templates in bitCRM enable automated and compliant WhatsApp communication. Use them for order updates, reminders, or promotions with dynamic variables for personalization. Follow these steps to create and customize your template.

## **Step 1: Go to the WhatsApp API Automation Template dashboard.**

## **Step 2: Click "Create new template".**

## **Step 3: Fill in the required template.**

## **Step 4: Fill in “Message” and input dynamic info like** `{{1}}` **and so on by clicking "Add variable \+" then select "Variable" dropdown. Read** **[How to Use Dynamic Variables in Automation Messages](https://knowledge.bitbybit.studio/bitcrm/how-to-create-a-personalized-message-using-dynamic-variables)** **for more details about using variables.**
