---
title: "How to Create a Personalized Message Using Dynamic Variables"
---

Using dynamic variables in bitbybit messages makes your outreach more personal and impactful. Customize messages based on customer data to improve engagement and relevance. Here’s a list of variables to help you get started.‍Customer Data:

1. Full Name:`{{customer.fullName}}` 
2. First Name:`{{customer.firstName}}` 
3. Last Name:`{{customer.lastName}}`
4. Email:`{{customer.Email}}` 
5. Phone:`{{customer.phone}}` 

Utilize these variables to create personalized messages that resonate with your customers.
