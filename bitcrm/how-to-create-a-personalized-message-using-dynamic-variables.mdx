## Customer Data Variables

Utilize these variables to create personalized messages that resonate with your customers:

- **Full Name:** `{{customer.fullName}}`
- **First Name:** `{{customer.firstName}}`
- **Last Name:** `{{customer.lastName}}`
- **Email:** `{{customer.email}}`
- **Phone:** `{{customer.phone}}`

Incorporate these variables into your message templates to dynamically insert customer-specific information, enhancing the personalization of your communications.
