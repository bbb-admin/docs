<Info>
**Prerequisites**
* **Meta Business Manager:** You must have Admin access.
* **Verification:** Your [Business Manager must be verified](https://business.facebook.com/settings/security) by Meta.
* **Phone Number:** A clean number that is **not** currently active on any WhatsApp App (Consumer or Business). If it is active, you must [delete the account](https://developers.facebook.com/docs/whatsapp/on-premises/get-started/migrate-existing-whatsapp-number-to-a-business-account) first.
</Info>

## Why use WhatsApp Cloud API

The Cloud API is the enterprise version of WhatsApp. Unlike the mobile app, it allows for:
* **High-volume broadcasting:** Send marketing campaigns to thousands of users.
* **Advanced Automation:** Build chatbots with buttons and list messages.
* **Multiple Agents:** Manage one number with a team using the bitbybit inbox.

    3. Navigate to **Accounts > WhatsApp Accounts**.
    4. Select your account and click **Payment Settings**.
    5. Add your credit card.
