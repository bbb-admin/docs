# How to Set Up Microsoft OAuth

Integrating Microsoft login into your application allows users to authenticate using their Microsoft accounts, providing a seamless and secure login experience. Follow these steps to set up Microsoft OAuth with bitLogin.

## Step 1: Access Microsoft Azure Account

- Visit the [Microsoft Azure Portal](https://portal.azure.com).
- Log in with your Microsoft account, or register if needed.

<img
  src="/images/azure-login.png"
  alt="Login to Microsoft Azure Portal"
  className="rounded-lg"
/>

## Step 2: Register Your Application

- In the Azure portal, navigate to **Microsoft Entra ID** > **App registrations**.
- Click **+ New registration**.
- Fill in the application details:
  - **Name**: Enter your store name.
  - **Supported account types**: Select **Accounts in any organizational directory and personal Microsoft accounts**.
  - **Redirect URI (optional)**: Choose **Web** and enter:
    ```
    https://api.bitbybit.studio/bitlogin/api/login/microsoft/callback
    ```
- Click **Register**.

<img
  src="/images/azure-register.png"
  alt="Register New Application"
  className="rounded-lg"
/>

## Step 3: Obtain App ID & App Secret

- After registration, go to **Certificates & secrets**.
- Under **Client secrets**, click **+ New client secret**.
- Add a description and set an expiration period (e.g., 365 days).
- Click **Add**.
- Copy the **Value** of the client secret immediately; you won't be able to see it again.

<img
  src="/images/azure-secret.png"
  alt="Add Client Secret"
  className="rounded-lg"
/>

## Step 4: Configure Microsoft OAuth in bitLogin

- Log in to your [bitLogin dashboard](https://app.bitbybit.studio).
- Navigate to **Settings** > **Microsoft OAuth**.
- Paste the **Application (client) ID** and **Directory (tenant) ID** from the Azure portal.
- Paste the **Client Secret** you obtained earlier.
- Click **Save Changes**.
- Enable the **"Use your store name on Microsoft login"** toggle until it turns green.

<img
  src="/images/bitlogin-microsoft.png"
  alt="Configure Microsoft OAuth"
  className="rounded-lg"
/>

## Step 5: Completion

You’ve successfully set up Microsoft OAuth. Securely store your **Client Secret**, as it’s crucial for secure authentication.
