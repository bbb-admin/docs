---
title: "How to Set Up Chat Widget Appearance"
---

Customizing your chat widget's appearance in bitChat enhances your store's user experience by aligning it with your brand’s identity. Make it more visually engaging and intuitive for your customers. Here’s how to set it up for optimal setting.

## **Step 1: Access bitChat Appearance Settings**

* Go to the [bitChat Chat Widget](https://app.bitbybit.studio/bitchat/wa-widget) Dashboard and click "Appearance."  


![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/66fe685d599329b4670f58f4_AD_4nXc08xYJ4iL9QVscOtzuy_-OLwJI4gjM6COiRu3iffXUTB92cSshK5d_tm8JhJzE7WvL5_2pm3_xSJo1SBm8dWuEe05lYb9C1ChpbI4h0grxDooW-RohkasLh443FCM5g7WB7p81oW59oBxASjaPHddqMis.png)Click "Appearance"‍

## **Step 2: Customize Button Widget**

* Choose button style and type the button text (optional).

## **Step 4: Set Widget Color**

* Choose between "Solid color" or "Gradient" for the widget background or you can customize by clicking the color box.  


![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/66fe685d69170883a5211b8f_AD_4nXe42IHw2cMnAVaxoun9KwNLG98v4TWFD0gX6JKTj5a994pcd8zGpHxxTe6yDI5smHE_JJAxTmbU_oRtM-lFMM6WymYvFpVOu_q8W9sV1F4YGRLXhsur8LALWKDzHWdWBTRa3tjuEKs_ocejzp_D8_PlNW39.png)Choose widget color‍
