---
title: "How to Enable Xendit Payment Link on bitbybit"
---

Easily integrate Xendit payment links with bitbybit to streamline transactions and automate payment processing. Follow this guide to set it up quickly:

#### **Step 1: Generate Your Xendit Secret Key**

**1. Log in** to your [Xendit Dashboard](https://dashboard.xendit.co/settings).  
  


2. Navigate to the **Settings** page and click on [**API Keys** under the **Developers** section](https://dashboard.xendit.co/settings/developers#api-keys).  


![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/67d90fe78eef684916cf9edf_AD_4nXcTPnhuZHK42K2T9OSztA-2bLKp9WMwvsMQoSfdnuAqWLRqvRk182083nx6Coj8qbWJWxM8JZAkLh4tsPG-v7GEuxXjISnlcQQUgDBZbOCxjgOUytbsnFtNpyL7oeF-ushiRTBktQ.png)3. Click **Generate secret key** and set the following permissions:  


![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/67d90fe6085b62e8458d3c9f_AD_4nXdLlw1IWOKYiBeGDBKg36nXaAPD5BehUumvdWKlkVpAnL1eqzFQJ5y_EWdYFGS-5sLr86JrRnZt7fFYHDUZIEFLmheOJ66cQnw3WCXODrE3vQEMgjhr0Eg0Uqo5u1xtkhEt7Yql.png)**● API Key Name**: Enter a name.

● **Report and Transaction Access**: Set to **Read**.

● **Other Permissions**: Set to **None**.  


![](https://cdn.prod.website-files.com/659655f5b20d395a5c69efcf/67d90fe78eef684916cf9ed6_AD_4nXd3StETUGFghu9IMnwgx7cShx9QHWqfyH6jvk6l12KAhdsJwI9fYwgp1PHeT-L0-_BKk0j0G4B7Nte7pD4PniBEegsJdlg9997eC-oh3aEkFzP8ldaLcGJVNjuqee3sVbEyIFYqsw.png)4. Click **Create Key**, then **copy and save** the secret key in a secure location.

## **Step 2: Integrate with Xendit**

* [Contact the **bitbybit** **team**](https://api.whatsapp.com/send/?phone=6281514231000&text=Hello+bitbybit+team%2C+I+need+some+assistance%0A%28https%3A%2F%2Fwww.bitbybit.studio%2Fknowledge%2Fbitchat%29%0A&type=phone_number&app_absent=0) to assist with the integration process.

## **Step 3: Automate Payment Link Generation**

* Once integrated, create an automation and select **“Xendit Payment URL”** as a dynamic variable to include the payment link in your workflow.
